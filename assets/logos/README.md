# Logos (Intro Conveyor)

Drop your logo images in this folder.

The intro conveyor tries these filenames (any of these extensions work: `.svg`, `.png`, `.webp`, `.jpg`, `.jpeg`):

- `kicad.*`
- `solidworks.*`
- `c-cpp.*`
- `java.*`
- `python.*`
- `reaper.*`
- `unity.*`
- `csharp.*`

If a logo file isn't found, the site will automatically fall back to text.
